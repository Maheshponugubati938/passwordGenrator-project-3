# Password Generator

A simple and responsive Password Generator built using HTML, CSS, and JavaScript.

## Features

* Generate secure random passwords
* Select character types:

  * Uppercase Letters (A-Z)
  * Lowercase Letters (a-z)
  * Numbers (0-9)
  * Special Characters (!@#$%^&*)
* Adjustable password length using a range slider
* Copy generated password to clipboard
* Reset all selections with a single click
* User-friendly interface

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)

## How to Use

1. Select the character types you want in your password.
2. Choose the password length using the slider.
3. Click the **Generate Password** button.
4. Click the **Copy** button to copy the generated password.
5. Use the **Reset** button to clear all selections.
